# SeleniumFramework
Framework for Selenium tests

### Command:
``` mvn clean test -Dsuites=suiteFile.xml -DappUrl=http://urlToApplication.com```

### Built with:

* Java
* Maven
* TestNG
* Selenium WebDriver
